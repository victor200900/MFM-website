# MFM Church Website

A dynamic church website built with **React.js** and **Tailwind**, showcasing events, sermons, and contact functionality.

## Demo

[Live Demo on Vercel ( I https://mfm-website-sand.vercel.app/)

## Features

- **Responsive Landing Page**: Welcoming interface built with React and Bootstrap.
- **Event & Sermon Previews**: Highlights upcoming events and recent sermons dynamically.
- **Contact Form**: Users can reach the church via a functional contact form.
- **Routing**: Smooth navigation using React Router.
- **Interactive UI**: Card carousel previews and modern design elements.

## Technologies Used

- React.js (Functional Components, Hooks)
- React Router
- Bootstrap 5
- CSS / Vanilla JS for interactivity
